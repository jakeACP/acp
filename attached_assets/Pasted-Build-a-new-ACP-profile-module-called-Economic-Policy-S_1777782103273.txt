Build a new ACP profile module called “Economic Policy Simulator.”

Purpose:
Let users redesign the federal budget by adjusting spending categories in a pie chart. The module starts with the current real-world U.S. federal budget, then lets users increase or cut categories based on their priorities.

Source baseline:
Use FY 2024 federal outlays as the starting dataset. CBO estimated total FY 2024 federal outlays at $6.752 trillion and a deficit of about $1.8 trillion. Use CBO as the preferred source for baseline updates. Sources: CBO Monthly Budget Review FY2024.

Total baseline should equal approximately $6.752 trillion.

Important:
Net Interest should be visible but locked by default because users cannot directly “choose” interest costs without changing debt, interest rates, or deficits. Add tooltip: “Interest is mostly determined by existing debt and interest rates, so it is locked in this simplified simulator.”

User Controls:
1. Interactive pie chart
- Show current federal spending as a pie chart.
- User can click a slice to select a category.
- Selected category shows a slider.

2. Category slider
Each category can be changed between:
-10% and +10%
Default: 0%

Example:
Defense baseline = $826B
User sets -10%
New amount = $743.4B

3. Overall spending slider
Add a separate “Total Spending Goal” slider:
Range: -10% to +10%
Default: 0%

This represents whether the user wants the whole federal budget to shrink or grow.

Behavior:
- Category sliders adjust individual categories.
- Overall spending slider adjusts the target total.
- Show whether the user’s category changes are above or below their selected total spending goal.
- If category choices do not match the target, show:
  “Your proposed category changes are $X above/below your total spending target.”

Budget Math:
baselineTotal = sum baseline amounts
targetTotal = baselineTotal * (1 + totalSpendingAdjustment / 100)
categoryAdjustedAmount = baselineAmount * (1 + categoryAdjustment / 100)
proposedTotal = sum categoryAdjustedAmounts
differenceFromTarget = proposedTotal - targetTotal

Display:
- Baseline total
- Proposed total
- Difference from baseline
- Difference from target
- Estimated deficit impact

Use FY 2024 baseline:
baselineOutlays = 6752
baselineReceipts = 4919
baselineDeficit = 1833

estimatedDeficit = proposedTotal - baselineReceipts

Results Screen:
Show:
- User’s proposed budget pie chart
- Baseline vs proposed comparison table
- Biggest increases
- Biggest cuts
- Estimated deficit or surplus impact
- “Your Budget Philosophy” summary

Budget Philosophy Labels:
- If total spending adjustment <= -5:
  “Spending Cutter”
- If total spending adjustment >= 5:
  “Public Investment Expander”
- If defense is cut and healthcare/social programs increased:
  “Domestic Investment Prioritizer”
- If defense increased and social programs cut:
  “Security & Austerity Prioritizer”
- If most sliders stay between -2 and +2:
  “Status Quo Budgeter”
- Otherwise:
  “Mixed-Priority Budgeter”
