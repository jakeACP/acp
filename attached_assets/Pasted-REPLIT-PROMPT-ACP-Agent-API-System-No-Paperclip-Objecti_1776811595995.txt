REPLIT PROMPT — ACP Agent API System (No Paperclip)

Objective:
Remove Paperclip (port 3001) and replace it with a secure, scalable API layer that allows external AI agents (Claw Machine / OpenClaw) to interact with ACP using role-based API keys.

🧹 1. Remove Paperclip بالكامل
Delete all Paperclip-related code, services, dependencies, and environment variables
Remove any references to:
Port 3001
Paperclip UI / routes
Agent orchestration logic tied to Paperclip
Clean up package.json and backend services
Confirm ACP runs cleanly on primary port only
🧠 2. Build “ACP Agent API Gateway”

Create a new backend module:

/server/api/agent/

Core responsibilities:

Authenticate agents via API keys
Enforce role-based permissions
Log all agent activity
Rate-limit and sandbox behavior
Return structured JSON responses optimized for AI agents

All endpoints must:

Require API key header
Validate permissions
Log request + response
Return structured JSON (agent-friendly)
